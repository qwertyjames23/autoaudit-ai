# AutoAudit.AI

A free AI-powered automation audit tool for businesses. Describe your workflow and get a personalized report showing exactly where automation can save you hours every week.

**Live:** https://autoaudit-ai.netlify.app

---

## What It Does

1. Business owner describes their daily workflow
2. AI analyzes the workflow and identifies automation opportunities
3. Report shows specific tools, solutions, and estimated time saved
4. CTA connects them directly with an automation specialist

## Stack

- **Frontend:** Plain HTML + Tailwind CSS (CDN) + Vanilla JS
- **Backend:** Netlify Functions (Node.js)
- **AI:** GPT-4o mini via OpenAI API
- **Lead Capture:** Formspree
- **Hosting:** Netlify

## Features

- 4-step guided wizard
- AI-generated automation report with tool recommendations
- Time savings estimate per opportunity
- Lead capture (email notification on every submission)
- WhatsApp CTA for direct consultation booking
- Mobile responsive
